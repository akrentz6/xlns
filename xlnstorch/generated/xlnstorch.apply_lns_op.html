xlnstorch.apply_lns_op

xlnstorch.apply_lns_op(torch_function, *args, **kwargs)

Performs the computation for the default LNS implementation of a given torch function in the logarithmic domain. This function is used to apply the LNS internal operation defined in HANDLED_FUNCTIONS.

Parameters:

  • torch_function (Callable) – The torch function for which the LNS operation is to be applied.

  • *args (Any) – Positional arguments to be passed to the LNS operation.

  • **kwargs (Any) – Keyword arguments to be passed to the LNS operation.

Returns:

The result of the LNS operation applied to the provided arguments.

Return type:

Any

Raises:

ValueError – If no implementations are registered for the given torch function. If no default implementation is set for the torch function. If the implementation key is not registered for the torch function.
